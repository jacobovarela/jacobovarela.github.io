"It's my website and I'll write what I want to."

In late 2020 I came across this post on HackerNews about building a website from scratch, with very basic HTML, CSS and Javascript. So I got inspired and decided I wanted to create my own portfolio website.

This is something that I've been planning to do since I finished my Data Science bootcamp back in 2018, but every time I got sidetracked because "you already have your CV and projects on LinkedIn, why would you spend time or money on this".

I now look at it as an opportunity to learn something new, this "project" forces me to dive into web development, code in a new language (javascript) and maybe explore a new framework (vue.js ?).

  • First thing I did was to buy the domain name through Namecheap.
  • I decided to use GitHub pages to host the website, I already know how to use the platform and it's easy to set up the custom domain name.
  • I soon learnt that the browser can't run Python, so in future iterations I am planning to use Netlify to host the website.
